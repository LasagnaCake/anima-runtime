First Steps

Before you can begin programming, you must first create a project.

For these tutorials, we will be using the concerto CLI.

First, via your command line of choice, enter the parent folder of your project.

Then, run the following command:

concerto create "<YOUR PROJECT NAME>" --type exe --lang breve

After that, your project should have the following folder structure:

/<YOUR PROJECT NAME>
  /src
    main.bv
  project.flow
  .gitignore
